本テーマについて

先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得することが目的であり手順書を読み実験を行い レポートなどにより報告する実験の一連の流れを実践できHTMLを用いたWebページ作成ができる。

プロジェクト課題について

私のプロジェクト課題は検索バーにお店のジャンルを入力し隣にある検索ボタンを押すことによって検索結果を下に表示するプログラムを書いた。 検索後他のお店を検索したい場合は検索バーの文字を消し新たにジャンルを入力することによって検索ができる。 他の検索を行った場合前の検索結果は自動的に消え新しい検索結果が出るようになっている。 またプログラムにないお店のジャンルを入力したりお店のデータがない場合は検索結果がありませんと表示するようになっている。

プロジェクト課題で作成したページ：project.html

数当てゲーム

答えの数字がランダムで決められており予想は固定されているリロードをすると答えが毎回ランダムに変わり外れれば予想した数字 より大きいか小さいかを教えてくれその結果はコンソールと実際のページ両方に表示される。 今回の課題でテキストボックスと検索ボタンを作成しテキストボックスに自分で数字を入れ検索ボタンを押すと自分が入力した数字で予想することができゲームをプレイすることができた。 したのコンソール画面には答えを残し表示することができた。 この数当てゲームでは1~10までの数字をコンピュータがランダムで一つ設定しその数字を自分たちが3回以内に予想し予想を3回間違えるか数字を当てることでゲームが終了するシステムになっている。

数当てゲームのページ:kazuate.html

各授業回の学習事項

第1回WWWとHTML

HTMLの基礎である文章の書き方や箇条書きの作り方、数字付きの箇条書きの仕方、文字のフォントサイズを大きくするやり方、文字にURLを設定する方法、画像の挿入の仕方、Githubへの保存の方法などを学んだ。

演習の成果：演習1-4:HTMLの様々な要素

第2回CSSによるWebページの装飾

HTMLをより魅力的なものにするために今回はHTMLの細かいところを変更するCSSを学んだCSSではフォントのサイズなどを変えるだけでなく背景や文字の色、URLを入れた文字の操作なども作って設定することができる。

演習の成果：演習Webブラウザ紹介

第3回JavaScriptの基本

JavaScriptはJavaに似ているところがあるが所々違うこともありJavaでは変数をそれぞれ指定しなければいけないところをJavaScriptはletの変数を宣言すればどんなものでも入れることができる比較では=を多くつけたり型変換は書くものが違ったりなど少し違う部分があり注意が必要である。

演習の成果：オブジェクトのメンバーを出力する

第4回DOMとJavaScript

HTMLからではなくjavaScriptからページの本文を変えるプログラムを書く練習をしたほかにもjavaScriptからcssと同じように書式や文字の色などを変えることができかなり便利である。 やり方はdocument.querySelector('')で場所を選択しdocument.createElement('')で要素を作りtextContent=''でテキストなどを挿入することができる他の方法で元々の情報を上書きをすることもできる。 javaScriptでテキストやcssを書き換える利点としては何かアクションを起こされ時に内容を書き換えることができるため何か選択して情報を出すサイトを作る時に便利である。

演習の成果：DOM練習

第5回イベント処理

今回のイベントの処理では検索欄を作成した。labelで検索キーのタイトルを作成しinputで検索欄を作り(テキストの検索欄だけでなくカレンダーや色、パスワードにもすることができる)buttonで検索を実行するボタンを作ることができる。 JavaScriptの方ではボタンなどにアクションがなされた時にどんな動きをするかを設定することができボタンを押すだけでなくカーソルが上に来たらやキーボードを押すことで作動したりなどさまざまなことができる。

演習の成果：イベントの種類

工夫点，苦心点，強調点，感想

この作成したページで工夫したところは背景をレンガ模様にし、本テーマについて、プロジェクト課題について、数当てゲーム、各授業回の学習事項の大きなコンテンツをそれぞれ枠で区切り各授業回の学習事項を他のコンテンツとは違う色にし授業の回のところを黒にしその回で学んだことのタイトルを赤色で表示し判別がしやすいようにした。 また後ろの背景も変えただけでなくスクロールとともに動かないようにするようにプログラムを書いた。 苦心点はいくつかありまずつまずいたところがCSSの組み方が難しくclassやIDなどを設定しそれぞれの色や大きさどのようにくくるかなどの設定にかなり手こずり思うように装飾できるまでにかなりの時間がかかった。 次につまずいたところはDOMの部分でありJavaScriptでHTMlで書いた構文を構文を作成するのに戸惑い場所を構文で設定しそこから使いたい属性などを設定して実行するシステムがかなり難しく感じ慣れるまで何回もやり直し時間がかかった。 次につまずいたところは最後の通信処理で通信の仕方は授業で習ったがそれを打ち込んだ文字からどのようにその情報が入っているデータに変換し表示させることに苦労した。 さらに新しく検索したときに表示されてるものを消すプログラムを探すのにも苦労した。 強調点はindexとprojectの装飾をこだわったモチーフにしたの某掲示板の装飾を意識して作った各授業回の学習事項以外を緑色にし各授業回の学習事項のところを白色にし第⚪︎回のところを赤にしその回のテーマを黒で書き某掲示板の感じを再現した。 さらに少し前に説明した各大きい項目のところ緑と白にしたがこの色は参考にした掲示板の色をネットで検索をかけ全く同じ色にした。 最後に作った感想は最初はHTMLを高校の頃授業で少し触ったことがありなんとなくでできるだろうと思っていたら最初からよくわからずCSSまでは苦戦はしたがなんとか食いついてこれたがJavaScriptの範囲が始まってからは何が何だかわからず先輩方や周りの人からヒントをもらい失敗しながらもなんとか食いつくことができた。 何回も失敗を繰り返している間に成功した時に達成感などを感じるようになりHTMLが面白いと感じるようになった。 周りの力はかなり借りてしまったが自分の力で簡単なサイトを作ることができると自信にもつながった今回習ったことは初歩的なもので将来的にはもっと本格的なものを作りたいと感じた。
